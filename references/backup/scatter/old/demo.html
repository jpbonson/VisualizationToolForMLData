Scatter-Matrix.js

Explore matrix data (samples as rows, observations as columns) using scatterplot matrix. Simply supply a CSV with a header. For example: 

<script type="text/javascript">
  var sm = new ScatterMatrix('http://my.domain/data.csv');
  sm.render();
</script>
